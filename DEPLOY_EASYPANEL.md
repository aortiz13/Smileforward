# Deploying Smile Forward to EasyPanel

> 📖 **La documentación completa de despliegue se ha movido al [README.md](./README.md).**
>
> Consulta la sección **[Despliegue en EasyPanel](./README.md#-despliegue-en-easypanel)** para instrucciones detalladas paso a paso.

## Resumen Rápido

1. Crear proyecto en EasyPanel → **+ Service** → **App** → **GitHub** → **Dockerfile**
2. Configurar variables de entorno (ver [README.md](./README.md#-variables-de-entorno))
3. Configurar puerto: `4550`
4. Deploy
5. Desplegar Edge Functions desde tu máquina local (ver [README.md](./README.md#-despliegue-de-edge-functions))
